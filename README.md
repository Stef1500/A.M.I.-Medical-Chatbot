# A.M.I. - Artificial Medical Intelligence

A Spring 2026 capstone project for Florida State University (**FSU**) course CEN4090L (**Software Engineering Capstone Project**). Developed by students: Stefano Buglione, Lauryn Brown, Marie Schwarz, and Nicolas Fernandez. These students are collectively known for this project as Team TheParents.

Artificial Medical Intelligence (**A.M.I.**) is a web-based AI medical virtual assistant designed for topics in Personalized Medicine.

Its goal is to provide source transparency in its responses. As process of an AI generating a response is considered "black box" in nature, **A.M.I.**  was created to combat this issue by providing the data (as sources) used to generate a response.
